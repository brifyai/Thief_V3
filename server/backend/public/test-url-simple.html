Test de URL - Versión Simple

Prueba cualquier URL de noticias automáticamente

Ejemplos: https://www.biobiochile.cl, https://www.emol.com/noticias/Nacional/
¿Cómo funciona?
  1. Ingresa la URL de la página de noticias
  2. Click en "Probar URL"
  3. El sistema detecta automáticamente las noticias
  4. Si encuentra noticias, puedes agregarla al sistema
Tip: Usa URLs de categorías o secciones específicas para mejores resultados.
